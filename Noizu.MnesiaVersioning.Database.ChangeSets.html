    noizu_mnesia_versioning v0.1.3 Noizu.MnesiaVersioning.Database.ChangeSets View Source

    The Changeset table is responsbile for tracking database changset status. Schema changes are packaged into changesets that include apply and rollback methods. When using Noizu Provided helpers the system is capable of generating the appropriate rolback steps.

    Select records in the table using an Exquisite query, see Exquisite.match/2 and mnesia:select.

    Options

    • limit - sets the count of elements to select in every continuation
    • lock - sets the kind of lock to use
    • select - Exquisite selector spec
    • qualified - whether to set a name for the record or not
